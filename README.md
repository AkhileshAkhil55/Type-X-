# ⌨️ TypeX – Typing Speed Test Application

A modern and responsive **Typing Speed Test** web application developed using **HTML, CSS, and JavaScript**. The application helps users improve their typing speed and accuracy by providing real-time performance metrics and multiple difficulty levels.

---

## 🌐 Live Demo

🚀 **Try the application here:**

**[https://akhileshakhil55.github.io/Type--X--](https://akhileshakhil55.github.io/Type-X-/)/**

---

## 📌 Features

- 🚀 Three Difficulty Levels
  - Beginner
  - Intermediate
  - Advanced

- 📊 Real-Time Statistics
  - Words Per Minute (WPM)
  - Accuracy
  - Time Elapsed
  - Score

- 🎯 Live Character Validation
  - Correct characters are highlighted.
  - Incorrect characters are marked instantly.

- 📈 Progress Bar
  - Tracks typing completion percentage.

- 🌙 Dark & Light Theme Toggle

- 📱 Fully Responsive Design
  - Works on Desktop, Tablet, and Mobile devices.

- 🔄 Restart Test Option

- 🏆 Result Summary
  - Displays Final WPM
  - Accuracy
  - Time Taken
  - Final Score
